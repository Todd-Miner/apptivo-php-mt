```

```

## Install

Via Composer

``` bash
$ composer require ToddMinerTech/ApptivoPhp
```

## Documentation
Generate documentation using phpDocumentator 3: https://www.phpdoc.org/

Sample command: php phpDocumentor.phar -d d:\www\apptivophp -f src -t documentation
